## System Online

Welcome to **Q:XS**, the Decentralized Intelligence Hub. This platform is engineered from the ground up for crypto-native researchers, traders, and builders who demand real-time data and zero fluff.

### What We're Building

- **Real-time Dashboard**: Live market data for BTC, ETH, SOL, XMR, and DOGE — streamed via WebSocket, rendered at the edge.
- **Research Reports**: Deep-dive analyses published daily, covering DeFi protocols, on-chain metrics, and market structure.
- **Blog**: Technical write-ups, system updates, and alpha leaks.

### Architecture

This site runs on **Astro 5** with hybrid SSR, deployed on **Cloudflare Pages**. Content is managed via Git-based Markdown — because the best CMS is `vim` + `git push`.

```bash
$ astro build && wrangler pages deploy ./dist
```

### What's Next

We're adding on-chain analytics, portfolio tracking, and automated report generation. Stay tuned.
